🔬 HTML-in-Canvas — Demo de integración

Esta página demuestra todas las funciones del módulo packages/excalidraw/htmlcanvas/.

1. Detección de soporte

Prueba isHtmlInCanvasSupported() + simulación para demostrar el fallback.

2. drawHtmlElement — renderizado nativo (simulado)

Sin API (fallback)

drawHtmlElement devuelve false, el canvas queda en blanco.

Con API simulada

Se inyecta drawElement en el contexto y se pinta un div HTML.

3. drawTextEditorInCanvas — textarea en canvas

Simula el editor de texto WYSIWYG pintado directamente en el canvas en vez de un overlay absolutamente posicionado.

4. drawHtmlElementWithEffects — composite + filter

multiply blend mode
blur filter

5. pasteHtmlToCanvas — paste de HTML rico

Pega HTML arbitrario directamente en el canvas (sin conversión a texto plano).

6. Clipboard — tipo "htmlContent"

Cuando la API está activa, parseHTMLFromPaste preserva el HTML raw como tipo "htmlContent" en vez de descomponerlo. 

Clic en "Simular paste" para ver el resultado
