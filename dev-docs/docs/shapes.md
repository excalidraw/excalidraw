# Supported Shapes

Excalidraw provides a set of built-in tools for drawing and annotating on the canvas. Each tool can be activated from the toolbar or with a keyboard shortcut.

## Drawing Tools

| Shape | Shortcut | Description |
| --- | --- | --- |
| Selection | `V` or `1` | Select, move, and resize elements on the canvas |
| Rectangle | `R` or `2` | Draw rectangular shapes |
| Diamond | `D` or `3` | Draw diamond (rhombus) shapes |
| Ellipse | `O` or `4` | Draw ellipses and circles |
| Star | `B` | Draw five-pointed stars |
| Arrow | `A` or `5` | Draw arrows that can bind to other shapes |
| Line | `L` or `6` | Draw multi-point lines and polylines |
| Draw | `P` or `7` | Freehand pencil drawing |
| Text | `T` or `8` | Place and edit text on the canvas |
| Image | `9` | Insert an image onto the canvas |
| Frame | `F` | Group elements into a named frame for clipping and export |

## Utility Tools

| Tool | Shortcut | Description |
| --- | --- | --- |
| Hand | `H` | Pan around the canvas by dragging |
| Eraser | `E` or `0` | Erase elements from the canvas |
| Laser Pointer | `K` | Ephemeral pointer trail, useful during presentations |
| Lasso | `V` or `1` | Freeform selection (when set as preferred selection tool) |
| Embeddable | — | Embed external content such as websites or videos |
