## 功能特性

Excalidraw 编辑器（npm 包）支持：

- 💯&nbsp;免费且开源。
- 🎨&nbsp;无限的基于画布的白板。
- ✍️&nbsp;手绘风格。
- 🌓&nbsp;深色模式。
- 🏗️&nbsp;可定制。
- 📷&nbsp;支持图片。
- 😀&nbsp;支持形状库。
- 🌐&nbsp;本地化（i18n）支持。
- 🖼️&nbsp;导出为 PNG、SVG 和剪贴板。
- 💾&nbsp;开放格式 - 将绘图导出为 `.excalidraw` JSON 文件。
- ⚒️&nbsp;广泛的工具 - 矩形、圆形、菱形、箭头、线条、自由绘制、橡皮擦...
- ➡️&nbsp;箭头绑定和带标签的箭头。
- 🔙&nbsp;撤销/重做。
- 🔍&nbsp;缩放和平移支持。

## Excalidraw.com

托管在 [excalidraw.com](https://excalidraw.com) 的应用是你可以用 Excalidraw 构建的最小展示。其[源代码](https://github.com/excalidraw/excalidraw/tree/master/excalidraw-app)也是本仓库的一部分，该应用具有以下特点：

- 📡&nbsp;PWA 支持（离线可用）。
- 🤼&nbsp;实时协作。
- 🔒&nbsp;端到端加密。
- 💾&nbsp;本地优先支持（自动保存到浏览器）。
- 🔗&nbsp;可分享链接（导出为只读链接与他人分享）。

我们将来会将这些功能作为 npm 包的即插即用插件添加。

## 快速开始

**注意：** 以下说明适用于在将 Excalidraw 集成到你自己的应用中时安装 Excalidraw [npm 包](https://www.npmjs.com/package/@excalidraw/excalidraw)。要在本地运行仓库进行开发，请参考我们的[开发指南](https://docs.excalidraw.com/docs/introduction/development)。

使用 `npm` 或 `yarn` 安装该包。

```bash
npm install react react-dom @excalidraw/excalidraw
# 或
yarn add react react-dom @excalidraw/excalidraw
```

查看我们的[文档](https://docs.excalidraw.com/docs/@excalidraw/excalidraw/installation)了解更多详情！
