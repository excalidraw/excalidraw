### Implementation notes
- `getScene`/`getUserId` read `excalidrawAPIRef`/`userIdRef` (updated every render) so the engine — constructed once
  per `[client, fileManager]` — never reads stale values. Engine `dispose()` on cleanup.
- Start/stop effect keyed on `[userId, fileManager]`: `engine.start(userId)` when signed in, `engine.stop()` on logout.
- Flush effect (`beforeunload`/`blur`/`visibilitychange==='hidden'` + flush-on-unmount), registered only when flag on.
- All hooks called unconditionally (Rules of Hooks); flag/client/user gating is inside the effects/memo. Flag-off and
  signed-out are safe no-ops; `syncNow()` returns `Promise.resolve()` and `notifyChange()` is inert when no engine.
