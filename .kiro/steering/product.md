# Excalidraw Product Overview

Excalidraw is an open-source virtual whiteboard for creating hand-drawn style diagrams and sketches. It's designed as both a standalone web application and a React component library that can be embedded in other applications.

## Key Features

- Hand-drawn aesthetic with sketchy, beautiful diagrams
- Real-time collaboration with end-to-end encryption
- Infinite canvas for unlimited drawing space
- Export capabilities (PNG, SVG, .excalidraw format)
- Extensible shape libraries
- Multi-language support (i18n)
- PWA support for offline functionality
- Wide range of drawing tools (shapes, arrows, free-draw, etc.)

## Architecture

The project consists of two main parts:

1. **Core Library** (`@excalidraw/excalidraw`): React component providing drawing functionality
2. **Web Application** (`excalidraw-app`): Full-featured web app at excalidraw.com with collaboration

## Target Users

- Developers integrating drawing capabilities into applications
- Teams needing collaborative diagramming tools
- Individuals creating wireframes, flowcharts, and technical diagrams
- Educational institutions for visual learning

## Notable Integrations

Used by major companies including Google Cloud, Meta, CodeSandbox, Notion, Replit, and many others as an embedded diagramming solution.
