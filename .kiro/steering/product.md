# Excalidraw Product Overview

Excalidraw is an open-source virtual whiteboard for creating hand-drawn style diagrams and sketches. It's designed as both a standalone web application and a React component library that can be embedded in other applications.

## Key Features

- **Hand-drawn aesthetic**: Creates beautiful diagrams with a sketchy, hand-drawn appearance
- **Collaborative**: Real-time collaboration with end-to-end encryption
- **Infinite canvas**: Unlimited drawing space for large diagrams
- **Export capabilities**: PNG, SVG, and native .excalidraw format support
- **Shape libraries**: Extensible with custom shape collections
- **Localization**: Multi-language support (i18n)
- **PWA support**: Works offline as a Progressive Web App

## Architecture

The project consists of two main parts:

1. **Core Library** (`@excalidraw/excalidraw`): React component that provides the drawing functionality
2. **Web Application** (`excalidraw-app`): Full-featured web app hosted at excalidraw.com with collaboration features

## Target Users

- Developers integrating drawing capabilities into their applications
- Teams needing collaborative diagramming tools
- Individuals creating wireframes, flowcharts, and technical diagrams
- Educational institutions for visual learning tools

## Integration Examples

Used by major companies including Google Cloud, Meta, CodeSandbox, Notion, and many others as an embedded diagramming solution.
